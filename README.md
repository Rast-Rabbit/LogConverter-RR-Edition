## 概要
このツールは、ココフォリアおよび Tekey で出力されたHTML形式のチャットログを整形し、より見やすく、共有しやすい形式に変換するためのWebアプリケーションです。  
キャラクターアイコンの設定、吹き出しの色やフォントのカスタマイズ、ログへの画像挿入、見出しの追加など、多彩な編集機能を提供します。  
編集した内容はプロジェクトファイルとして保存・読み込みが可能で、最終的なログはZIP形式、またはHTML単体としてエクスポートできます。

## 派生版で追加・変更した点（主な差分）
- 発言者名をクリックした際に、**発言者を変更可能**に
- 「タブ切り替え」操作で、チャットが属する**タブを変更可能**に
- チャットを上下に移動して**並べ替え可能**に  
  - 編集画面上で、ひとつ上/下の要素と表示順の内部値を入れ替え
- **編集領域のサイズを拡大**
- **HTML単体保存機能**を追加  
  - CSSおよび画像データをHTML内に埋め込み、HTMLファイル1つだけで配布・閲覧可能
- メッセージ削除時に**セパレーターが残る不具合を修正**

## 主な機能
* **ログファイルの読み込み**
  * ココフォリア形式のHTMLログ
  * Tekey形式のHTMLログ (v2形式推奨)
  * 本ツールで保存したプロジェクトファイル (`.cclogproj`)
* **キャラクター設定**
  * 発言者ごとの表示名、デフォルトアイコン、向き（左右）、テーマカラー、文字色の設定
  * 表情差分アイコンの登録と、メッセージごとの適用
  * 新規キャラクターの追加
* **表示カスタマイズ**
  * 吹き出しの色（左右別）、フォントサイズ、ログエリア背景色、アイコンサイズ、吹き出し最大幅の調整
  * フォントファミリーの選択（Noto Sans JP、Inter、Noto Serif JP、M PLUS Rounded 1c、各種システムフォント）
  * ログ表示エリアの高さ調整
  * 基本文字色、文字の縁取り色の設定
  * 出力ログ用の背景画像設定（ぼかし・明度補正付き）
  * アイコン下に名前を表示するモードの切り替え
* **ログ編集機能**
  * タブおよび発言者によるログフィルタリング
  * ログの先頭への画像挿入
  * メッセージごとの表示モード切替（吹き出し/地の文）
  * メッセージごとのアイコン変更（デフォルト/表情差分/個別アップロード）
  * メッセージごとの向き変更（キャラクター設定を上書き）
  * メッセージ内容の直接編集
  * 任意の箇所への画像挿入（キャプション付き）
  * 任意の箇所への見出し挿入（H1～H6レベル）
  * 任意の箇所への新規発言追加
  * 不要なメッセージや画像の削除（削除確認の省略オプションあり）
  * （派生版）メッセージの並べ替え、発言者変更、タブ変更
* **プロジェクト管理**
  * 編集中の状態をプロジェクトファイル (`.cclogproj`) として保存・読み込み
  * キャラクター設定や表示カスタマイズ設定のブラウザLocalStorageへの一時保存・読み込み
* **エクスポート**
  * 整形・編集済みログをZIP（HTML/CSS/画像一式）で出力
  * （派生版）整形・編集済みログを**HTML単体（CSS・画像埋め込み）**で保存
  * 出力時のHTMLタイトル、ZIPファイル名のカスタマイズ

## 使い方
1. **ファイルの準備**
   * ココフォリアまたはTekeyからチャットログをHTML形式でエクスポートしておきます。
2. **ツールの起動**
   * `index.html` をウェブブラウザで開きます。特別なサーバー環境は不要です。
3. **ログの読み込み**
   * 画面上部の「ココフォリアログ選択」「Tekeyログ選択」ボタンからログファイルを読み込みます。
   * または、以前このツールで保存したプロジェクトファイル (`.cclogproj`) を「プロジェクト(.cclogproj)選択」ボタンから読み込みます。
4. **キャラクター設定**
   * 「キャラクター設定」タブを開き、発言者ごとに表示名、アイコン、向き、テーマ色、文字色などを設定します。
   * 必要に応じて表情差分アイコンを登録したり、新規キャラクターを追加したりします。
5. **表示カスタマイズ**
   * 「表示カスタマイズ」タブを開き、フォント、各種色、サイズなどを調整します。
6. **ログの編集**
   * ログ表示エリアで編集（削除/挿入/見出し追加/画像挿入/モード切替/並べ替え等）を行います。
7. **保存・エクスポート**
   * 編集途中の状態は「プロジェクトを保存 (.cclogproj)」で保存できます。
   * 完成したログは「整形済みログをZIPで出力」または「整形済みログをHTMLで保存」で出力できます。
